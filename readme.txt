Simple and lightweight plugin which integrates Boardea.com video storyboards with Wordpress

== Description ==

### Plugin integrates Boardea.com video storyboards with Wordpress

Plugin filters out URLs of YouTube videos and adds links to video storyboard (+ basic statistics) generated by [Boardea.com](https://www.boardea.com/). No more plain links to videos you have no idea what they are about. 

The plugin makes no permanent changes to the content (it does make no change to the database). 

See demo [here](https://www.boardea.com/demo).

### Bug reports

Bug reports are welcomed over classic WordPress [plugin support page](https://wordpress.org/support/plugin/boardea-storyboard-integration) or directly over Boardea [contact](https://www.boardea.com/contact) form.

== Installation ==

Classic Wordpress installation.

== Frequently Asked Questions ==

= Does your plugin store anything in database? =

No, it doesn't. All is generated on the fly. Plugin makes no permanent changes to the content.
